# Installation

## Install the package

```bash
pnpm add -D @orbisk/vue-use-media-recorder
```

### Nuxt

- Auto import for Nuxt is supported. Just add the module to your `nuxt.config`:

```ts
export default defineNuxtConfig({
  modules: ['@orbisk/vue-use-media-recorder/nuxt']
})
```
